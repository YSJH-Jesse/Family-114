# 臺南市立鹽水國民中學 114學年度家庭教育成果網

本網站為 **臺南市立鹽水國民中學輔導組** 114學年度學校家庭教育實施成果網頁，專為提供評鑑委員及親師生線上瀏覽建置。

## 📌 成果內容收錄 (38 項主題)

1. **到校宣講與教師研習**：家庭教育輔導團宣講、性別平等教育研習與入班宣導、網路安全與心理健康講座。
2. **倫理教育與祖孫週**：國一至國三倫理教育微電影欣賞（《爺爺2.0》、《機器人與孤獨的婆婆》等）。
3. **節慶與感恩活動**：母親節卡片與立體櫥窗展示、祖父母節感恩卡片與《祖孫面對面》學習單、愛家傳真情活動。
4. **影片融入教學**：家庭教育影片融入學習單（《鞋子不見了》、《珍惜身邊的他》、《愛使我們更靠近》、《阿嬤的衛生紙》）。
5. **校本自編議題教材**：自編學習單（《甜蜜的家庭》、《家庭金主張-模擬家庭會議》、《餐桌上愛的時光》）。
6. **各領域議題融入**：國文、英語、數學、自然、科技、社會、健體、藝文全領域課程融入。
7. **親職教育與親師座談**：小團體輔導活動、親職教育講座與親師座談會。
8. **學生繪本得獎與榮譽牆**：
   - 臺南市 110 年學校家庭教育學生主題創作徵選 小書繪本**特優**（鄭天立）
   - 臺南市 110 年度校園 CEDAW 日繪畫甄選 **甲等**（何青薌）
   - 臺南市 110 年度校園 CEDAW 日繪畫甄選 **佳作**（溫青烜、吳婕菱）
   - 臺南市 111 年度校園 CEDAW 日繪畫甄選 **佳作**（李宜軒）
9. **社區生活營與心理健康**：個別諮商親職講座、療程式小團體、心能力 2.0 幸運憂轉。
